# Happy API Starter

happypeter 的  REST API 脚手架代码。适合使用 es6/express/mongodb 的你。

### 功能

- CORS 支持通过 [cors](https://github.com/troygoode/node-cors)
- 通过 [body-parser](https://github.com/expressjs/body-parser）实现 req.body 解析
